# Risk & Portfolio Management

В данном разделе представлены глубокие исследования в области количественных финансов (Quantitative Finance) и теории управления портфелем. Основной акцент сделан на математической оптимизации активов с учетом рыночных ограничений, ликвидности и транзакционных издержек.

**Ключевой стек:** `Python`, `MOEX ISS API`, `NumPy`, `Pandas`, `SciPy (Optimization)`, `Matplotlib`, `PCA`.

---

## Featured Project: Оптимизация портфеля российских акций (2015–2025)
**Исследование эффективных границ и лимитов риска**

Масштабный проект по построению и стресс-тестированию инвестиционных портфелей, основанный на данных 30 наиболее ликвидных акций Московской биржи за 10-летний период.

* **[portfolio_management.ipynb](./portfolio_management(project).ipynb)** — Аналитический движок проекта: сбор данных через API, расчет ковариационных матриц, квадратичное программирование.
* **[strategy_presentation.pdf](./portfolio_management(pres).pdf)** — Презентация стратегии, визуализация эффективных границ и выводы по риск-менеджменту.

### Глубокий анализ и методология:
1. **Сбор и предобработка данных (Data Engineering):**
    * Автоматизированный парсинг исторических котировок через **MOEX ISS API**.
    * Борьба с **Survivorship Bias** (ошибкой выжившего): строгий отбор компаний с полной историей торгов для корректности ковариационных матриц.
    * Обработка корпоративных действий (сплиты, например, Транснефти) для устранения искусственных разрывов цен.

2. **Математическая оптимизация (Modern Portfolio Theory):**
    * Построение классической границы Марковица и поиск портфеля минимальной волатильности (**Global Minimum Variance Portfolio**).
    * Решение задач оптимизации с жесткими ограничениями: отсутствие коротких позиций (Long-only), лимиты на долю одного эмитента (Compliance limits).
    * Использование **Adjusted Betas по Блуму** для корректировки прогнозов риска и учета возврата к среднему.

3. **Учет рыночного трения (Implementation Shortfall):**
    * **Net Frontier Analysis:** Уникальный расчет эффективной границы с учетом транзакционных издержек и проскальзывания. 
    * Доказательство гипотезы о том, что избыточный оборот портфеля (turnover) нивелирует математическое преимущество агрессивных стратегий.

4. **Анализ факторов риска (Risk Decomposition):**
    * Применение **PCA (Метода главных компонент)** для декомпозиции риска портфеля. Выявление скрытых факторов, доминирующих в движении российского рынка.
    * Использование моделирования **Монте-Карло** для оценки устойчивости портфеля к экстремальным рыночным шокам.

## Ключевые компетенции:
* **Asset Allocation:** Построение оптимальных весов с учетом индивидуальных риск-профилей.
* **Quantitative Risk Research:** Понимание ограничений модели Марковица (Error Maximization) и поиск путей их регуляризации.
* **Backtesting:** Проверка гипотез на длительных исторических горизонтах с учетом реалистичных комиссионных планов.
